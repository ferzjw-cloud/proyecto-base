# Maquetación UI estilo WhatsApp — Entrega

### 2. Descripción breve
Este proyecto consiste en la maquetación en HTML y CSS de una interfaz tipo WhatsApp. La maqueta incluye un sidebar con lista de chats y un panel principal con la conversación. Solo se utilizaron HTML y CSS, sin JavaScript.

### 3. Estructura del proyecto
/index.html  
/main.css  
assets (1.gif)

### 4. Cómo visualizarlo
En VSC abrir ejecutar index.html y ver en cualquier navegador.

### 5. Decisiones de diseño / Consideraciones
Los **colores** elegidos en algunos casos solo fueron para dar mejor contraste o por considerar una combinación más atractiva.
La **tipografía** usada es la sugerida en clase
Se practico el uso de **flexbox** a fin de organizar elementos y que estos se adapten a la ubicación y el espacio requerido.
**Grid** en este caso para organizar el contenido de la app en dos columnas y distintas filas.

*Se definieron variables para colores y al contar con distintas clases por separado para distintos elementos se ajustó el estilo a cada uno.
